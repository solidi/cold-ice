Cold Ice GoldSrc Edition

This project is a rebuild of Cold Ice 1.75, a popular Half-Life mod back in 1999.

Features
- Offhand Grappling Hook - "mp_grapplinghook" - "+hook"
- Vest Device
- Silenced Handgun
