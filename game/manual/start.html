START UP A SERVER, JOIN A COLD ICE GAME

 

Step me through the process of setting up a server!

After you installed the cold ice files, simply follow these directions to start up a server.

1) Start up the game "Half Life" as normal.
2) Select the "custom game" text on the menu
3) Select the MOD named "Cold Ice" and hit the button, "Activate"
4) Go back to the main menu and select "multiplayer"
5) Select the menu text named "internet games" or if you own a lan system and have buddies wating with cold ice already installed, choose "lan game"
6) Select the menu text named "create game"
7) From here you can change variables and other misc things before you launch the game.
8) FRAG NASTY! :)

This version includes a map named "Stalkyard2" and you should play it... its excellent.

 

Step me through the process of joining a cold ice server!

After you installed the cold ice files, simply follow these directions to join a cold ice server.

1) Start up the game "Half Life" as normal.
2) Select the "custom game" text on the menu
3) Select the MOD named "Cold Ice" and hit the button, "Activate"
4) Go back to the main menu and select "multiplayer"
5) Select the menu text named "internet games" or if you own a lan system and have buddies wating with cold ice already installed, choose "lan game"
6) Refresh the list and filter the menu with the game name "ice"
7) Join a game that has the description "Cold Ice 1.7". DONOT join a server with an older version and/or later version.
8) If you experince major lag, its probably due to your internet settings and/or system. Try to tweak the settings with the command "rate". (ex. rate 9999 for cable users, 3500 for 56k users, 2000 for 28k users)
9) HAPPY CAPPIN' =)
