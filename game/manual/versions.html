Version Confliction.

As noted on the page of connection to a server, this mod is version sentitive. If you recieve a cl_dll wrong version message when trying to connect to an cold ice server, this means that the server is runninng a older or newer version of cold ice. Please stop by our site periodically to find out what version is currently out.
