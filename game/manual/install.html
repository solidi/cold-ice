INSTALLATION

 

Step me through the process!

After you simply download the zip file , follow these instructions.

1) Unzip the files and directory to a temporary folder (X:/unzipped/coldice)
2) Go to the directory known as "half-life" in the sierra folder
3) In the "half-life" folder create a new folder named "ice"
4) Take the files in the temporary folder and dump them into the "ice" folder (Meaning, ALL FILES!).

Thats all there is to it. It should take you no more then 3 minutes. (15 seconds for professinals) =)
