Current Problems with Cold ice.

 

1) If you are having trouble seeing the action because of the HUD system , and runnin on a low
resolution, delete the gfx.wad file in the ice folder to decrease the front size.
2) If you are having trouble with your visuals, cant get the console to scroll down, or your controls
are screwed up, simply delete the files "config.cfg" and "settings" files in the ice folder
3) If you are running under software mode for visual, and you cant see the scoreboard, download
the new gfx.wad that corrects this problem. If you still cant see the scoreboard and/or running
a different vis mode, simply bind a key to the command "+showscores" (Applies to Version 1.6)
4) If you can not start a server because of "map unknown not found", simply type "map XXXX"
XXXX would stand for any map you have in the map directory of the valve folder, or any of the
valve maps.
5) Remember, Cold ice only works with version 1.0.1.0 or higher. Any lower and the MOD will not
work.
6) MISSING ADPTI, or anything to that effect means that the .dll is missing (duh!) or is installed in the
wrong directory. The dll should be in the directory of "dll" in the ice folder.
7) Cold ice is for WIN32 systems only! Theres no consideration for a linux port.
8) Cold ice should be installed in the directory of "Half-life" in a folder named "Ice"
9) To make this MOD appear, simply go to the custom game menu and double click on "Cold Ice"

 

Anymore issues that we dont know about, please contact us here, so we can post the information
