DO NOT DELETE THIS DIRECTORY!
IF DELETED, THE LOGGING SYSTEM WILL NOT LOAD AND RECORD YOUR STATS.
YOU MAY, HOWEVER, DELETE OR MOVE YOUR LOG FILES. (.html files)

IF A LOG IS PRERSENT IN THIS DIRECTORY AND HL-PRO LOGS A FILE THAT HAS 
THE SAME NAME (stats_game(number).html), IT WILL BE OVER-WRITTEN.

REMEBER, THE LOG FILES ARE NUMBERED BY THE CURRENT GAME OF THE SERVER.
IT COULD BE ANY NUMBER UP TO 32 CHARACTER SPACES.
